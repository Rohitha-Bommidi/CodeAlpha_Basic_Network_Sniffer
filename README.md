# # Basic Network Sniffer

## Objective
Capture and analyze network traffic packets using Python and Scapy.

## Features
- Captures network packets
- Displays Source IP Address
- Displays Destination IP Address
- Identifies Protocols (TCP, UDP, ICMP)
- Displays Payload Data

## Tools Used
- Python
- Scapy
- VS Code

## Output
Successfully captured and analyzed network packets.
